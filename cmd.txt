Основные команды CMD
Команда	Описание

cd	Изменяет текущую директорию.
cls	Очищает экран консоли.
dir	Показывает содержимое директории.
echo	Выводит текст на экран.
exit	Закрывает командную строку.
find	Ищет строку в файле.
ipconfig	Показывает настройки сетевого подключения.
ping	Проверяет доступность узла в сети.
mkdir	Создает новую директорию.
rmdir	Удаляет директорию.
del	Удаляет один или несколько файлов.
copy	Копирует файлы из одного места в другое.
move	Перемещает файлы или переименовывает их.
tasklist	Показывает список запущенных процессов.
taskkill	Завершает процесс по его имени или идентификатору.
shutdown	Выключает или перезагружает компьютер.
chkdsk	Проверяет диск на наличие ошибок и исправляет их.
format	Форматирует диск или раздел.
systeminfo	Показывает информацию о системе.


Сетевые команды
Команда	Описание

netstat	Показывает активные соединения и порты.
nslookup	Запрашивает информацию о DNS для указанного домена.
tracert	Отслеживает маршрут до указанного IP-адреса или домена.


Дополнительные команды
Команда	Описание

reg	Управляет реестром Windows (чтение, запись, удаление).
robocopy	Расширенная команда для копирования файлов и директорий.
Этот список включает в себя основные команды, которые могут быть полезны для работы с командной строкой в Windows 1235.
